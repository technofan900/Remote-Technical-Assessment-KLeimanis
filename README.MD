# PHP CRUD Prototype

## Features
- CRUD for Projects
- CRUD for Issues
- MySQL database
- PDO prepared statements
- Basic input validation

## Setup

1. Configure DB:
   - config/database.php
   - database should be created automatically

2. Run:
   - Start Apache/MySQL
   - Run command:
     php -S localhost:8080

## Technologies
- PHP 8
- MySQL
- PDO
